<a name="setup"/>

## Setup

First **fork** the project repository, then clone the project onto your local machine:
```
$ git clone git@github.com:**YOUR USERNAME**/Googly-Notes-TS-React.git
$ cd Googly-Notes-TS-React
$ npm install
$ npm start
```
